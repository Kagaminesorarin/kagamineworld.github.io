在Python中如何以绝对路径或者相对路径导入文件

绝对路径：绝对路径就是文件的真正存在的路径，是指从硬盘的根目录(盘符)开始，进行一级级目录指向文件；

相对路径：相对路径就是以当前文件为基准进行一级级目录指向被引用的资源文件。
